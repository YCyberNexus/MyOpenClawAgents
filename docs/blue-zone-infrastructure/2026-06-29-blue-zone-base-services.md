# 蓝区基础服务梳理

本清单梳理蓝区需补齐的基础服务，按 12 类组织，逐项标注作用、参考实现与优先级。共 12 类、77 项。

优先级口径：**P0** 平台运转或安全合规的前提；**P1** 打通研发交付流水线所需；**P2** 治理与稳定性增强。

## 分类总览

| # | 类别 | 涵盖 |
|---|---|---|
| 1 | 跨区数据通道与制品摆渡 | 外部资源单向、受控进入蓝区，及跨区数据流转、外界大模型访问与出站管控 |
| 2 | 源码托管与协作 | 代码仓库、评审、大文件、检索 |
| 3 | 制品与依赖仓库 | 容器镜像 + 各语言包私服 + 系统包镜像 |
| 4 | 构建与持续集成 | CI、构建缓存、动态测试环境、制品签名 |
| 5 | 计算与存储底座 | 容器编排、对象存储、数据库、缓存、配置 |
| 6 | 大模型与智能服务底座 | 外界大模型访问管控 + 内网推理（降级备份）、网关、向量检索、GPU 调度 |
| 7 | 密钥、证书与身份 | 密钥管理、内网 CA、统一身份、签名准入 |
| 8 | 监控、扫描与审计 | 可观测、漏洞与代码扫描、审计留痕 |
| 9 | 数据安全与合规 | 加密、防泄漏、出域管控、合规基线 |
| 10 | 备份、容灾与高可用 | 备份、冗余、灾备恢复、回滚 |
| 11 | 网络与平台基础服务 | DNS、时间同步、负载均衡、网络隔离 |
| 12 | 运维接入与配置管理 | 堡垒机、配置即代码、变更管理、资产台账 |

## 1. 跨区数据通道与制品摆渡

| 编号 | 服务 | 作用 | 参考实现 | 优先级 |
|---|---|---|---|---|
| 1.1 | 入站制品摆渡通道 | 外部容器镜像 / 依赖包 / 系统包 / 模型权重 / 工具，经受控单向通道同步入网，含申请—审批—安检—入库台账闭环 | 网闸 / 数据二极管 + 镜像与包同步工具 | P0 |
| 1.2 | 出站管控网关 | 控制蓝区对外数据流向，默认拒绝、白名单放行 | 单向网关 + 出站策略引擎 | P0 |
| 1.3 | 跨区数据流转通道 | 受控的跨网络区消息 / 数据中继 | 受控反向代理 + 协议白名单 | P1 |
| 1.4 | 入站内容安全检查 | 入站制品的病毒 / 恶意包 / 供应链投毒检测与来源核验 | Trivy / ClamAV + 哈希签名核验 | P1 |
| 1.5 | 跨区代码单向同步 | 蓝区代码向外层网络区单向镜像，外层只读 | git mirror over 单向网关 | P1 |
| 1.6 | 外界大模型访问通道 | 蓝区访问外部大模型服务的受控专用出站通道，经大模型网关统一收口并做 DLP 与审计 | 专用出站链路 + 大模型网关(6.2) + 出站 DLP | P0 |
