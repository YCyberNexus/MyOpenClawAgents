# 蓝区所需基础服务清单

## 1. 跨区通道与外部资源受控导入

| 编号 | 服务 | 说明 | 优先级 |
|---|---|---|---|
| 1.1 | 外部资源受控导入通道 | 容器镜像、语言包、系统包、模型权重、工具等外部资源受控进入蓝区 | P0 |
| 1.2 | 出站管控网关 | 控制蓝区对外数据流向，默认拒绝、白名单放行 | P0 |
| 1.3 | 跨区数据流转通道 | 蓝区与黄区/绿区之间的受控消息、回调与数据中继 | P1 |
| 1.4 | 导入内容安全检查 | 外部导入资源的病毒、恶意包、供应链投毒检测与来源核验 | P1 |
| 1.5 | 跨区代码与制品只读同步 | 蓝区代码和制品向隔离区域形成只读镜像 | P1 |
| 1.6 | 外部大模型访问通道 | 蓝区访问外部大模型服务的受控专用通道 | P0 |
